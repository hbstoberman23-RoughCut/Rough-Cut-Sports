ROUGH CUT SPORTS — BUILD 1

Features:
- Baseball, fastpitch, and slowpitch event setup
- Team and payment tracking
- Automatic round-robin scheduling
- Mobile game scoring and innings
- Baseball/fastpitch pitch counts
- Slowpitch home-run tracking
- Automatic standings and seeding
- Expense and profit tracking
- Backup and restore
- Installable offline web app support

Open index.html to test on a computer.
For phone installation, upload all files to an HTTPS host.
iPhone: Safari > Share > Add to Home Screen.
Android: Chrome > Install App / Add to Home Screen.
